## Requirements

- Currently only working on Iphone 6 
- Xcode 6
- iOS 8.0+
- If you have troubles with the compilation try to clean the DerivedData Xcode directory: `rm -rf ~/Library/Developer/Xcode/DerivedData/`

## Installation

- Download the project
- pod install
